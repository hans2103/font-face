# prepare fonts
- Download fonts via [http://google-webfonts-helper.herokuapp.com/fonts](http://google-webfonts-helper.herokuapp.com/fonts)
- Convert to base64 encrypted [https://jaicab.com/localFont/](https://jaicab.com/localFont/)
